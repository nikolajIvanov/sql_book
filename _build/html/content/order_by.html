SQL ORDER BY#

Das ORDER BY-Schlüsselwort wird verwendet, um das Ergebnis in aufsteigender oder absteigender Reihenfolge zu sortieren.

Syntax#

SELECT column1, column2, ...
FROM table_name
ORDER BY column1, column2, ... ASC|DESC;
